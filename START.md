# 喵了个咪 — 桌面悬浮窗猫咪宠物

## 启动方式

```bash
cd ~/Desktop/喵了个咪
npx electron .
```

> 首次运行需等 Electron 下载（约 30 秒），之后秒开。

## 操作指南

| 操作 | 效果 |
|------|------|
| 拖拽猫头 | 移动到屏幕任意位置 |
| 双击猫头 | 拎出交互模式 → 漫画气泡聊天 |
| 右键猫头 | 菜单（朋友圈/好友/日程） |
| 输入文字 → 发送 | AI 对话（硅基流动 LLM） |
| 点 🎬 | 自动演示 4-5 轮对话 |
| 点 ⋯ | 菜单 → 朋友圈/好友/日程 |
| 30连击 | 隐藏彩蛋 |
| 闲置 5-20 分钟 | 猫咪自己出来溜达 |

## AI 配置

已接入硅基流动 LLM：
- 日间(6:00-18:59)：活泼可爱猫咪模式
- 夜间(19:00-5:59)：猫咪哲学模式

API Key 存在 `llm-key.txt` 中。

## 多人交互测试

打开 `reminder-demo.html`（日程弹窗演示）
打开多个猫窗口 → 好友模块 → 互相发送指令
支持：🎮 多选动作指令 / 💬 传话 / ⏰ 定时传话
