## 📁 文件放这里

| 文件夹 | 放什么 | 格式 |
|--------|--------|------|
| `animations/` | Lottie 动画 JSON 文件 | `.json` |
| `images/` | 静态图（猫头、泡泡、头像等） | `.png` / `.jpg` |
